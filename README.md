# Cypress Login Tests

Projeto de automação de testes E2E utilizando Cypress.

## Cenários Automatizados

- Login com sucesso
- Senha inválida
- Usuário inválido
- Logout com sucesso

## Tecnologias

- Cypress
- JavaScript
- GitHub

## Objetivo

Projeto desenvolvido para estudos de automação de testes e composição de portfólio QA.

## Como executar

npm install

npx cypress open
