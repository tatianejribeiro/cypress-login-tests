# Cypress Login Tests

Projeto de automação de testes End-to-End (E2E) utilizando Cypress.

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar automação de testes web utilizando Cypress, aplicando conceitos de validação funcional e testes de autenticação.

## Cenários Automatizados

* ✅ Login com sucesso
* ✅ Senha inválida
* ✅ Usuário inválido
* ✅ Logout com sucesso

## Tecnologias Utilizadas

* Cypress
* JavaScript
* GitHub

## Estrutura do Projeto

```text
cypress-login-tests
│
├── README.md
├── package.json
├── cypress.config.js
│
└── cypress
    ├── e2e
    │   └── login.cy.js
    │
    └── support
        └── e2e.js
```

## Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o Cypress:

```bash
npx cypress open
```

3. Selecione o teste:

```text
login.cy.js
```
